# **Welcome**
+ Hi! Welcome to LFYS's first 3d game engine!
+ LFYSG3D is a lightweight, Java-based 3D software rasterization engine built from scratch. Designed for learning and prototyping, it runs anywhere Java runs â€” no GPU required.

  - For linux debian
    + Terminal
      ```bash
      #you only have to do this part once
      sudo apt install xorg default-jdk
      cd LFYSG3D
      chmod +x run
      #this is the file to run the code
      ./run
      ```
   - For windows
      + CMD
        ```cmd
          cd C:\Users\YourUsername\OneDrive\Desktop\LFYSG3D\
          run.bat
        ```
     + Powershell
       ```powershell
         cd C:\Users\YourUsername\OneDrive\Desktop\LFYSG3D\
         Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
         .\run.ps1
       ```
    
- And that's it! :)
